üîô Connection to Achievement 1

This Django project builds directly on the foundation from Achievement 1, where a command-line Recipe App was created using core Python. The concepts carried forward and evolved:

  • Data Models: The `Recipe` class from OOP became the Django `Recipe` model.
  • Database: Raw SQL and SQLAlchemy were replaced by the powerful and integrated Django ORM.
  • Logic: Python functions that handled user input are now implemented within Django Views that handle web requests.
